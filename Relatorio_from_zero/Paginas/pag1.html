Para verificar a sua compreensão de uma troca do ARP, desenhe uma figura que mostre a solicitação ARP e os pacotes de resposta enviados entre seu computador e o gateway padrão. Faça isso para o caso que examinamos de seu computador fazendo um ARP para o gateway padrão. Identifique um pacote com a solicitação e o outro com a resposta. Indique o sender e target MAC e endereço IP para cada pacote; você pode usar o Wireshark para inspecionar os pacotes para obter esses valores.
